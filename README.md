# SMS System Counter-Strike
A SMS system for purchasing vip privileges for Counter-Strike 1.6 servers.

### Back-end Features:
- FastRoute
- PDO
- SSL Ready
- MVC Structure

### Front-End Features:
- Responsive
- W3C Valid
- Bootstrap last build
- HTML5
- Font Awesome
- CSS 2/3 (strict)
- JQuery
